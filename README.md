# Randomized MPC py

Randomized Model Predictive Control with python

## How to install 

1. Download

```bash
git clone git@github.com:kohonda/randomized_mpc_py.git
cd randomized_mpc_py
```

2. Create venv

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install

```bash
cd randomized_mpc_py
pip3 install wheel
pip3 install -e .
```
